Allows user to place a link to their Google Helpouts listing through a widget on their website.

== Description ==

Google Helpouts is a service from Google where anyone can get help from anyone on a number of topics from different providers.
Those providers often need to promote their services on their personal WordPress sites and need a widget to facilitate sharing Helpout links.

== Installation ==

1. Upload `/helpouts-widget` folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and find the widget called Google Helpouts Listing Promotion.
